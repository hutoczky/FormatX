Biztonságos workflow

A FormatX Suite Pro minden kritikus művelet előtt megjeleníti a kiválasztott eszköz nevét, azonosítóját, méretét, modelljét és a művelet következményét. A veszélyes műveletek előtt kötelező a kézi megerősítés.

A folytatáshoz írd be:

TÖRLÉS /dev/sdb

Ez segít megelőzni a véletlen adatvesztést és átláthatóvá teszi a rendszerkarbantartási folyamatot.

Élő rendszeradatok

A FormatX Suite Pro rendszerfelügyeleti modulja valós, élő rendszeradatokat használ. A CPU, RAM, hálózat, tárhely, USB eszközök, hőmérsékleti adatok és Bazzite-specifikus állapotok a rendszerből kerülnek kiolvasásra. Ha egy adat nem érhető el, a program N/A vagy Nem elérhető állapotot jelenít meg.

Cross-platform és hordozható működés

A FormatX Suite Pro célja, hogy Windows, Linux / Bazzite és macOS rendszereken is használható legyen. A hordozható telepítő képes pendrive-ra, külső SSD-re vagy egyéni célmappába is települni.

A felhasználó csak három fő indítót lát:

  • Start-FormatX-Windows.exe
  • Start-FormatX-Linux.sh
  • Start-FormatX-macOS.command

Az indítók automatikusan felismerik a rendszer és a CPU architektúra típusát, például x86_64 vagy ARM64 / aarch64, majd a megfelelő buildet indítják.
