Adatvédelem

A nyilvános FormatX weboldal nem kér és nem tárol bankkártyaadatot, licenckulcsot vagy kliensoldali hozzáférési tokent.

A weboldal működése

A megjelenési mód választását a böngésző helyi tárhelye őrizheti meg. Ez az adat az adott böngészőben marad, és nem kerül a FormatX szerverére.

A legújabb kiadás megjelenítéséhez az oldal egy alkalommal lekéri a nyilvános GitHub Releases metaadatot. A kéréshez a weboldal nem használ tokent. A GitHub a saját adatvédelmi feltételei szerint kezelheti a hozzá érkező hálózati adatokat.

Letöltés és hibajelentés

A programcsomag a GitHub Releases szolgáltatásból tölthető le. A visszajelzési űrlap nem küld adatot FormatX backendnek: a megadott szövegből előkészíti a GitHub nyilvános hibajegyoldalát, ahol a beküldés külön történik.

A GitHubon beküldött hibajelentésekre a GitHub adatvédelmi szabályai vonatkoznak. Hibajelentésben ne ossz meg jelszót, privát kulcsot, tokent, teljes licenckulcsot vagy más bizalmas adatot.
